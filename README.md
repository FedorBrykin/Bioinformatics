# HW1: Парное выравнивание ортологичных генов

## Фенотип

**Название:** Глазо-кожный альбинизм I и II типов (Oculocutaneous Albinism, OCA1/OCA2)  
**OMIM:** [#203100 (OCA1)](https://omim.org/entry/203100), [#203200 (OCA2)](https://omim.org/entry/203200)

**Описание:**  
Глазо-кожный альбинизм — группа аутосомно-рецессивных наследственных заболеваний, обусловленных нарушением синтеза меланина в меланоцитах кожи, волос и глаз. OCA1 вызывается мутациями в гене *TYR*, кодирующем фермент тирозиназу, которая катализирует ключевые этапы меланогенеза. OCA2 обусловлен мутациями в гене *OCA2*, кодирующем мембранный транспортный белок меланосом, регулирующий pH органеллы и тем самым активность тирозиназы. Клинические проявления обоих типов схожи: белая или слабо пигментированная кожа, светлые волосы, голубые или серые радужины, нистагм, фотофобия и сниженная острота зрения.
