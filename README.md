# Множественное выравнивание гомологов гена TP53

## 1. Выбранный ген

**Gene Symbol:** `TP53`  
**Полное название:** Tumor protein p53 (клеточный опухолевый антиген p53)  
**Организм:** *Homo sapiens*  
**RefSeq белка:** NP_000537.3  
**OMIM:** [*191170](https://omim.org/entry/191170)  
**NCBI Gene:** [7157](https://www.ncbi.nlm.nih.gov/gene/7157)

TP53 — ген-супрессор опухолей, кодирующий транскрипционный фактор p53 («страж генома»). Белок активируется в ответ на повреждение ДНК и запускает остановку клеточного цикла, репарацию ДНК или апоптоз. Мутации в TP53 обнаруживаются более чем в 50% всех злокачественных опухолей человека.

## 2. Параметры BLAST

Поиск гомологичных последовательностей проводился в два этапа.

**Инструмент:** NCBI BLAST — blastp (белок против белка)  
**Запрос:** NP_000537.3 (белок TP53 человека, 393 а.о.)

| Параметр | Значение | Обоснование |
|----------|----------|-------------|
| Program | blastp | белковые последовательности консервативнее нуклеотидных — позволяет найти гомологов у рыб, амфибий и рептилий |
| Database | refseq_protein | только проверенные RefSeq-записи, без дублей и синтетических конструктов |
| Exclude organism | *Homo sapiens* (taxid: 9606) | исключение запросного вида |
| Include organism | Vertebrata (taxid: 7742) | ограничение позвоночными |
| Exclude organism | Mammalia (taxid: 40674) | исключение млекопитающих для поиска далёких гомологов |
| Max target sequences | 100 | |
| E-value threshold | 0.001 | стандартный порог значимости |
| Matrix | BLOSUM62 | стандартная матрица для белковых выравниваний |
| Gap open / extend | 11 / 1 | значения по умолчанию |

Млекопитающие (мышь, макак-резус) были добавлены отдельным поиском по базе `refseq_select_prot` для таксономического разнообразия набора.

## 6. Объединяющий таксон

**Латынь:** *Vertebrata*  
**Русский:** Позвоночные  
**Ранг:** Подтип (Subphylum)

Позвоночные — подтип типа Хордовые (Chordata), объединяющий все организмы с позвоночным столбом или его хрящевым аналогом. Включает около 66 000 известных видов: хрящевые рыбы (акулы, скаты), костные рыбы, амфибии, рептилии, птицы и млекопитающие. Все 13 видов данного исследования — от акулы до человека — входят в подтип Vertebrata, что делает его наименьшим общим таксоном для данного набора. Сохранность гена TP53 во всех этих группах (~48–100% идентичности белка) свидетельствует о том, что функция p53 как «стража генома» возникла уже у общего предка позвоночных и поддерживается эволюционным отбором на протяжении более 450 миллионов лет.
